# LinkNote 範例索引

最方便的使用方式是在 App 左側按「？ 使用說明與範例」，再按「建立完整範例包」。系統會建立以下五篇筆記，而且不會覆蓋已存在的同名筆記。

## 1. 睡眠與學習的 Knowledge Graph

示範：

- 主詞、關係、受詞
- 因果與正負影響
- 多條路徑共同指向學習成效

核心關係：

```text
規律作息 → 睡眠品質 → 注意力 → 課堂理解 → 學習成效
睡眠品質 → 記憶鞏固 → 長期記憶 → 學習成效
睡眠不足 → 疲勞 → 注意力下降
```

## 2. 地方博物館展覽心智圖

示範：

- 核心主題
- 展示內容
- 觀眾體驗
- 執行工作
- 多層縮排條列

## 3. 圖書館借書申請流程圖

示範：

- 開始與結束
- 館藏是否可借
- 借閱資格是否有效
- 預約與補資料的返回流程

## 4. 生物分類樹狀圖

示範：

- 動物界與植物界
- 脊椎動物、無脊椎動物
- 種子植物、非種子植物
- 節點收合與展開

## 5. 咖啡店數位轉型策略地圖

示範完整的策略因果鏈：

```text
員工數位工具能力 → 點餐與出餐流程 → 顧客等待感 → 回購率 → 營收
銷售資料能力 → 菜單推薦內容 → 個人化與便利性 → 回購率
```

每個目標都含 KPI，可直接點擊節點修改。
