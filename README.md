# LinkNote v2.1

LinkNote 是一個可部署到 GitHub Pages 的純前端離線筆記 App。

## 主要功能

- Markdown 編輯與預覽
- `[[雙向連結]]`、反向連結與標籤
- **目前筆記內的知識圖譜**
- 由 Markdown 標題與縮排清單自動產生的心智圖
- 可編輯策略地圖：四構面、KPI、因果箭頭與 SVG 匯出
- IndexedDB 儲存與舊版 `localStorage` 自動搬移
- JSON 合併匯入，不整批覆蓋現有筆記
- 垃圾桶、刪除復原與永久刪除
- 每篇筆記最多 30 個版本紀錄
- 完整 JSON 備份、Markdown 匯出與備份提醒
- PWA、iPhone 加入主畫面與離線快取

## v2.1 修正

- 修正心智圖與策略地圖按鈕點擊後沒有反應的問題。

- 更新 Service Worker 快取版本，避免 GitHub Pages 繼續載入舊 JavaScript。

## 筆記內知識圖譜格式

一般 Markdown 標題與條列會自動成為概念節點：

```markdown
# 多語言學習系統

## AI 功能
- 生成式 AI
  - 即時回饋
  - 個人化建議
```

要明確指定概念關係，可在同一篇筆記加入 `knowledge` 程式區塊：

````markdown
```knowledge
生成式 AI | 提供 | 即時回饋
即時回饋 | 提升 | 學習成效
學習分析 -> 支援 -> 個人化學習
知識圖譜 → 概念連結
```
````

支援格式：

- `主詞 | 關係 | 受詞`
- `主詞 -> 關係 -> 受詞`
- `主詞 → 受詞`，此時關係預設為「關聯」
- `主詞 --關係--> 受詞`

按右上角 `◎` 後，會建立目前筆記的知識圖譜，箭頭上會顯示關係名稱。

## 心智圖格式

心智圖讀取目前筆記的 Markdown 標題與縮排清單：

```markdown
# 研究主題

## 文獻回顧
- 生成式 AI
- 知識圖譜
  - 概念節點
  - 關係邊

## 系統實作
- 前端
- 資料庫
```

按右上角 `⌘` 開啟心智圖。

## 策略地圖操作

1. 開啟一篇筆記，按右上角 `⇧`。
2. 選擇構面、輸入目標並按「新增目標」。
3. 點選目標後，按「編輯所選」加入 KPI。
4. 按「建立連線」，依序點選起點與終點。
5. 可重新命名四個構面。
6. 每篇筆記保存自己的策略地圖。

## 更新既有 GitHub 專案

1. 先在舊版 LinkNote 匯出一份完整 JSON 備份。
2. 用本資料夾的檔案覆蓋 GitHub 儲存庫中的同名檔案。
3. Commit 並 Push 到 `main`。
4. 等待 GitHub Pages 重新部署。
5. 首次開啟新版時重新整理頁面一次；若 iPhone 主畫面仍顯示舊版，完全關閉 App 後再開啟。

## 合併匯入規則

- 不會先清空現有筆記。
- 以相同 ID 或同名標題判斷為同一篇。
- 內容完全相同時不重複加入。
- 其中一份內容包含另一份時保留較完整內容。
- 內容不同時，在原筆記下方加入「匯入合併內容」區段。
- 策略地圖目標與因果箭頭也會一起合併。

## 資料安全限制

IndexedDB 與持久儲存權限能降低瀏覽器自動清理資料的機率，但使用者若主動清除 Safari 或 Chrome 的網站資料，本機筆記仍可能被刪除。請定期將完整 JSON 備份存到 iCloud Drive、Google Drive 或電腦。
