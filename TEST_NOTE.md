# 多語言 AI 學習流程測試

## 系統模組

- 學習者介面
  - AI 對話
  - 單字本
  - 學習報告
- AI 服務
  - 即時回饋
  - 個人化推薦
- 資料層
  - 語言知識庫
  - 學習紀錄

```knowledge
生成式 AI | 提供 | 即時回饋
即時回饋 | 提升 | 學習成效
學習分析 | 支援 | 個人化推薦
個人化推薦 | 促進 | 持續學習
```

```flow
開始((開始)) -> 設定目標[設定學習目標]
設定目標 -> 程度測驗[進行程度測驗]
程度測驗 -> 判斷{是否達到基礎程度？}
判斷 ->|是| 推薦[推薦適性教材]
判斷 ->|否| 基礎[複習基礎內容]
基礎 -> 程度測驗
推薦 -> 練習[進行 AI 對話練習]
練習 -> 報告[產生學習報告]
報告 -> 完成((完成))
```
